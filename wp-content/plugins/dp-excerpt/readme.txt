An advanced post excerpt plugin for WordPress.

== Description ==

DP Excerpt is a WordPress plugin, it can generate a better excerpt from content. 

1. Keep HTML markup in the excerpt(Optional)
2. Keep Shortcode output in the excerpt(Optional).
2. Cut off the excerpt to a given length using either character count or word count.
3. Customize the excerpt length and the ellipsis character.
4. Automatically add a "read more" link after your excerpt, linking to the full post.
5. Can ignore `<!--more-->` tag was used before the summary of a specified length.
6. Can ignore manual excerpt.
7. The real excerpt length, HTML markup and shortcode output is not counted.

== Installation ==

1. Upload the entire folder `dp-excerpt` to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Change Settings in 'Settings > DP Excerpt' under 'Setting' Menu in your WordPress admin panel.
