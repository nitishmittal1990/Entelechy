== Description ==

This plugin gives options to display "Table of contents" container on your wordpress post or page. Many variety of easing animations supported.
You can also choose when you would like the Scroll to top becomes active.
Its very easy to theme and use.
You can enable or disable this plugin just by setting 'Enabled' under Settings > JCWP Table of contents > Enabled to "No"

== Installation ==

1. Upload unzipped plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the "JCWP Simple TOC" Options on left configure your preferences.
